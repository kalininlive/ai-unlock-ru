# AI SERVICE UNBLOCK

**AI SERVICE UNBLOCK** — это современное браузерное расширение для Chrome, предназначенное для удобного обхода региональных блокировок популярных нейросетевых сервисов. 

## Описание
Многие AI-сервисы ограничивают доступ пользователям из определенных стран. Это расширение позволяет автоматически направлять трафик этих сервисов через прокси, используя технологию PAC (Proxy Auto-Configuration), что обеспечивает высокую производительность и минимальное влияние на остальной трафик.

Данное приложение не осуществляет обход ограничений доступа, установленных государственными органами Российской Федерации, включая Роскомнадзор.

Функциональность приложения направлена исключительно на обеспечение доступа к сервисам, доступ к которым может быть ограничен правообладателями или поставщиками услуг (в частности, Google и Anthropic) по географическому признаку. На момент разработки и распространения приложения указанные сервисы не включены в перечни запрещённых ресурсов на территории Российской Федерации.

В целях соблюдения требований действующего законодательства Российской Федерации, приложение реализует механизм периодической проверки актуальных реестров ограничений доступа, публикуемых Роскомнадзор. В случае внесения соответствующих сервисов в указанные реестры, их дальнейшая обработка и маршрутизация через приложение автоматически прекращается.

Таким образом, приложение функционирует в рамках действующего правового регулирования и не предназначено для обхода установленных государством ограничений доступа.

## Основа проекта
Данный проект основан на кодовой базе [Aimagine-life/gemini-unblock](https://github.com/Aimagine-life/gemini-unblock). Мы переработали интерфейс, структуру и добавили поддержку множества дополнительных сервисов.

## Основной функционал
- **Разблокировка AI сервисов**: Gemini, ChatGPT, Claude, Poe, Perplexity и другие.
- **Премиальный UI**: Современный дизайн в стиле "Glassmorphism" с плавными анимациями.
- **Мультиязычность**: Поддержка русского и английского языков (RU | EN).
- **Масштабируемость**: Легкое добавление новых сервисов через конфигурационный файл.
- **Умное проксирование**: Проксируются только запросы к AI-сервисам.

## Технологии
- Manifest V3
- Vanilla JS & CSS
- Chrome Proxy API
- Declarative Net Request (для спуфинга заголовков)

## Установка и настройка
1. Скачайте репозиторий: `git clone https://github.com/kalininlive/ai-unlock-ru`.
2. Откройте `chrome://extensions/` в браузере Chrome.
3. Включите "Режим разработчика" (Developer mode).
4. Нажмите "Загрузить распакованное расширение" (Load unpacked) и выберите папку `extension`.
5. Настройте свой прокси в интерфейсе расширения.

## Где купить прокси?
Для стабильной и быстрой работы расширения требуются качественные прокси. Мы рекомендуем сервис **Proxy6**:
🔗 **[Купить надежные IPv4/IPv6 прокси](https://proxy6.net/a/79776)**

## Архитектура
Расширение состоит из:
- **Background**: Мониторинг вкладок и управление прокси-настройками.
- **Popup**: Пользовательский интерфейс с динамическим списком сервисов и настройками.
- **Lib**: Модули для работы с PAC-скриптами, хранилищем и иконками.
