**AI SERVICE UNBLOCK** — это высокотехнологичное браузерное расширение, разработанное для обеспечения бесперебойного доступа к мировым AI-сервисам и профессиональным инструментам. Забудьте о блокировках и наслаждайтесь скоростью работы через ваш собственный прокси в премиальном интерфейсе.

## ✨ Ключевые преимущества

### 💎 Премиальный дизайн (Glassmorphism)
Мы создали интерфейс, который вдохновляет. Плавные анимации, эффект «матового стекла» и глубокая цветовая палитра делают использование расширения эстетически приятным.

### 🚀 Масштабируемость и кастомизация
- **Динамический список ресурсов**: Добавляйте любые сайты одним кликом. Расширение автоматически подтянет иконку (favicon) и создаст индивидуальный переключатель.
- **Умное управление**: Активируйте прокси только для тех сервисов, которые вам нужны в данный момент.

### 🔐 Безопасность и конфиденциальность
- **Proxy Masking**: Ваши учетные данные надежно скрыты в интерфейсе. Пароли отображаются только при редактировании.
- **Изолированный трафик**: Проксируются только выбранные домены. Основной интернет-серфинг идет напрямую, сохраняя максимальную скорость.

### 📦 Предустановленные пресеты (v1.1.0)
- **Google Ecosystem**: Полная поддержка всех сервисов Google (Ads, Cloud, Workspace, Analytics, YouTube, News и др.).
- **AI Leaders**: OpenAI (ChatGPT & Platform), Anthropic Claude, Perplexity, Grok, Sora, Quora Poe.
- **Google Auth**: Встроенная поддержка авторизации через аккаунты Google.

## 🛠 Установка

1. Скачайте репозиторий.
2. Откройте `chrome://extensions/`.
3. Включите **Режим разработчика** (Developer mode).
4. Нажмите **Загрузить распакованное расширение** (Load unpacked) и выберите папку проекта.
